## How to run

1. ```git clone https://github.com/MatheusCandido1/salary-suggestor.git```

2. ``` cd benefitter-challenge ```

3. ``` npm install ``` or ``` yarn ```

4. ``` npm run dev ``` or ``` yarn dev ```
